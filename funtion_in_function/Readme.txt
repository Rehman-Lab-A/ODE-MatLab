Steps to Run the Code
1. Open MATLAB on your computer.

2. Run the main script by typing the following in the MATLAB command window:
   main

3. The results will be displayed in the MATLAB command window.

        To modify parameters, open mainScript.m in the MATLAB editor, 
 change the values of a11, a12, a21, a22, d1, or d2,
 save the file, and rerun mainScript.
